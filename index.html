Communication Engineering Tool

This is an online tool developed to help students understand the fundamentals of communication systems engineering. It is used in Communication Systems I course in the Department of Electrical and Computer Engineering at the University of the West Indies. It is a front end app developed in Angular, stylized using Angular material and deployed in Heroku. Plots were done using Plotly.js.
Tech Used: Angular, Plotly.js, Heroku

PON Planning Tool

This is an online tool developed to help students understand the fundamentals of passive optical network planning. It is used in Transmission Systems course in the Department of Electrical and Computer Engineering at the University of the West Indies. It is a front end app developed in Angular, stylized using Angular material and deployed in Heroku. Plots were done using Plotly.js. Map data was obtained from Mapbox. Routing data was obtained from OSRM (Open Source Routing Machine).
Tech Used: Angular, Plotly.js, Mapbox, OSRM, Heroku
URL:PON Planning Tool

LOS Microwave Tool

This is an online tool developed to help students understand the fundamentals of line of sight microwave network planning. It is used in Transmission Systems course in the Department of Electrical and Computer Engineering at the University of the West Indies. It is a front end app developed in Angular, stylized using Angular material and deployed in Heroku. Plots were done using Plotly.js. Map data was obtained from OpenStreetMap. SPLAT! source code (RF Signal Propagation, Loss, And Terrain analysis tool) was translated to Angular to obtain elevation data.
Tech Used: Angular, Plotly.js, Heroku
URL:LOS Microwave Tool

VSAT Planning Tool
